## 论文概要

- **核心论点：** 大语言模型擅长反应式决策（检索→生成），但战略传播需要预测式决策（枚举→推演→比较→选择）。当前以"内容生产"为中心的 AI 应用思路错置了重点。
- **理论锚点：** Fuerth 前瞻性治理（Anticipatory Governance），替代 Nye 三层公共外交
- **案例：** 美国对华芯片禁令叙事战（2022-2026），聚焦 NVIDIA 黄仁勋与"互相伤害"叙事
- **字数：** ~16,000 字
