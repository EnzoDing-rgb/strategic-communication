## 核心论点（一句话）

战略传播要求传播者预测受众反应并据此选择策略，但当前 multi-agent 仿真缺少前向世界模型——本文提出将 World Model 架构引入战略传播 agent，弥合"反应"与"预测"之间的计算鸿沟。

## 前置知识：核心概念解释

### 1. 生成式 Agent（Generative Agents）

来源：Park et al. (2023), "Generative Agents: Interactive Simulacra of Human Behavior"（斯坦福小镇）。

**是什么：** 用大语言模型（LLM）驱动的软件 agent，能模拟类人日常行为——起床、做饭、上班、聊天、交朋友。

**怎么工作（四个组件）：**

| 组件 | 做什么 | 人话类比 |
|------|--------|----------|
| Memory Stream（记忆流） | 把所有经历用自然语言存下来，带时间戳 | 你的日记本 |
| Retrieval（检索） | 根据 recency（最近发生）+ importance（重要程度）+ relevance（和当下相关）从记忆中捞相关内容 | 你回想"上次和他聊过什么" |
| Reflection（反思） | 定期从大量记忆中提炼高层洞察 | 你睡前总结"我最近是不是太忙了" |
| Planning（规划） | 把目标分解成具体行动步骤 | 你今天早上在脑子里排今天的待办清单 |

**核心特征：** Agent 的行为由**过去的记忆**驱动。Agent 不预测未来，它检索过去。

**关键局限（论文自己承认的）：**
- 记忆检索不完整——记得要去 party，不记得被邀请过
- 幻觉添油加醋——声称某人要做宣布，实际没有
- LLM 参数知识泄露——agent 用模型预训练知识填补 gap
- 指令调优导致过度合作——从不拒绝建议

**但论文没指出（这正是我们论文的核心发现）的局限：**
- Agent **不预测**自己行为的后果
- Agent **不模拟**"如果我这样说，对方会怎么反应"
- Agent **不优化**传播策略——它只是在做"看起来合理的下一步"，不是在做"最有效的下一步"

### 2. 世界模型（World Model）

**不是大众理解的"世界模型"（Sora 那种生成视频的）。** 我们说的是计算神经科学和 AI 研究中的世界模型：

**是什么：** Agent 内部维护一个对环境的预测性表征。Agent 不只感知当前状态，还学习"如果我做 X，接下来会发生 Y"。用这个预测来选择最优动作。

**LeCun 的 JEPA 架构（人话版）：**
- 传统 AI：看到图 A → 输出标签 / 下一句话
- World model AI：看到状态 S → 预测如果采取动作 A，状态会变成 S' → 选择能让 S' 最接近目标的那个 A

**核心区别：**

| | 生成式 Agent | World Model Agent |
|---|---|---|
| 时间方向 | 回顾（查记忆） | 前瞻（预测结果） |
| 决策依据 | "之前类似情况我做了什么" | "这个动作会导致什么后果" |
| 策略优化 | 无（做合理的事） | 有（选效果最好的动作） |
| 对他人建模 | 无显式心智理论 | 可学习"他人如何响应我的动作" |

### 3. 约瑟夫·奈的公共外交三层框架

Joseph Nye 是美国国际关系学者，"软实力"概念的提出者。他把公共外交分为三层：

| 层次 | 做什么 | 时间尺度 | 例子 |
|------|--------|----------|------|
| 第一层：日常沟通 | 对事件做实时反应 | 小时/天 | 使馆发言人回应新闻、外交部推特 |
| 第二层：战略传播 | 围绕核心叙事持续传播 | 月/年 | 围绕一个主题做一整年的宣传活动 |
| 第三层：关系建设 | 建立长期人际/机构联系 | 年/十年 | 奖学金、学术交流、文化项目 |

**为什么和我们的论文相关：**

- 第一层（日常沟通）是**反应性的**——事件发生→回应。生成式 agent（检索记忆→生成回应）天然适合建模这一层。
- 第二、三层（战略传播、关系建设）是**预测性的**——你必须提前想"一年后目标受众会变成什么样"，然后倒推今天该做什么。这需要 world model。

**Nye 没明确说过"outcome prediction"，但他对第二、三层的描述隐含了这个要求：**
> 公共外交的更高层次"不是对事件的反应，而是通过一套简单、一致的信息来塑造受众的长期认知"（Nye, 2008）。

"塑造长期认知"意味着你必须有一个关于"受众认知如何随时间变化"的模型——这正是 world model。

**我们的学术贡献：** 首次将 world model 计算框架与 Nye 公共外交理论做形式化对齐。Nye 说了 what（战略传播应该做长期认知塑造），我们补上了 how（需要一个预测性世界模型才能做到这件事）。

## 第一章：引言

### 1.1 研究背景

- 战略传播是国家安全的核心工具——不是军事硬实力，而是通过信息、叙事和价值观竞争在国际体系中获取影响力
- 数字化、AI 技术正在重塑战略传播的战场：信息传播速度、精准度、个性化程度前所未有
- 学术界出现两种回应：一种警惕 AI 的风险（deepfakes、虚假信息），一种探索 AI 的工具价值（计算社会科学、multi-agent 仿真）
- 本文属于第二种路径：不是讨论 AI 带来什么威胁，而是讨论 AI 仿真**缺少什么能力**才能成为战略传播的有效研究工具

### 1.2 问题陈述

- 当前最先进的 multi-agent 仿真（以 Stanford 生成式 agent 为代表）能产生丰富的 emergent social behavior——传播信息、建立关系、协作行动
- 但这些 agent 的架构是**反应性的**：观察→检索记忆→生成反应。它们不预测行为的后果
- 战略传播的学术本质是**效果预测 + 策略选择**：不只是在合适的时候说合适的话，而是从多种可能的传播策略中选择能最有效塑造受众认知的那个
- 这个 gap 不是工程问题——是**架构假设的差异**

### 1.3 研究问题

本文围绕一个核心问题展开：

> 当前的生成式 agent 架构在什么意义上不足以建模战略传播？World model 的计算框架能提供什么？

拆解为三个子问题：
1. 生成式 agent 的架构假设（记忆驱动、反应式决策）与战略传播的要求（效果预测、策略选择）之间有何结构性差距？
2. World model 的计算框架（前向预测、潜在表征学习、基于模型的规划）如何对应战略传播的核心任务？
3. 这种架构升级对国家安全研究有何方法论含义？

### 1.4 论文结构与贡献

**贡献声明：**
1. 首次识别生成式 agent 架构在战略传播建模中的结构性局限（不是性能不够好，是架构假设不对）
2. 提出 World Model 增强的战略传播 agent 框架，弥合"反应"与"预测"之间的计算鸿沟
3. 以 Nye 三层公共外交框架为锚，论证架构升级的理论必要性，并给出可检验的未来实验蓝图

**论文结构：** 第二章打基础 → 第三章找差距 → 第四章提框架 → 第五章聊安全 → 第六章画蓝图 → 第七章收尾

## 第二章：理论基础

本章分三个理论支柱：生成式 agent 架构、世界模型计算框架、Nye 公共外交理论。

### 2.1 生成式 Agent 的架构逻辑

**核心架构拆解（Park et al., 2023）：**

- Memory Stream：自然语言记忆对象（observation + reflection），带创建时间戳和最近访问时间戳
- Retrieval Function：score = recency（指数衰减 0.995）+ importance（LLM 打分 1-10）+ relevance（embedding 余弦相似度）
- Reflection：当最近事件 importance 之和超过阈值 150 时触发，LLM 生成高层问题→检索相关记忆→提炼洞察并引证
- Planning：日议程→小时级→5-15 分钟级，自顶向下递归分解
- Reaction：每时间步判断"继续计划"还是"响应新观察"，对话时检索与互动对象的记忆

**这个架构的哲学假设：**
- 合理的行为来自于对相关过去经验的充分检索和上下文整合
- 决策质量取决于记忆检索质量 + LLM 推理能力
- 行动可以被描述为一连串可分解、可执行的具体步骤

**隐含假设（论文未明说但我们从中发现）：**
- Agent 不需要预测行为的后果——它只需要"在相似情况下做合理的事"
- 环境对 agent 行为的反馈通过观察→记忆→反思的闭环间接实现
- "策略"是不必要的——好的行为自然涌现于好的记忆和反思

### 2.2 世界模型：从反应到预测

**什么是世界模型（计算层面的定义）：**
世界模型是 agent 内部对环境的预测性表征。形式化地说：给定当前状态 s 和候选动作 a，世界模型输出对结果状态 s' 的预测。Agent 通过"心理模拟"不同动作的结果来选择最优动作。

**LeCun 的 JEPA 及其前提：**
- 传统监督学习：学习从输入到输出的映射（分类、回归）
- 自监督学习（SSL）：学习输入的内部表征，不依赖标签
- JEPA（Joint Embedding Predictive Architecture）：在表征空间中做预测，不是在原始数据空间做预测。不预测每一个像素，而是预测"发生了什么"

**为什么预测在表征空间中做：**
- 原始数据（文本、像素）包含大量不相关的细节
- 表征空间只保留对决策有意义的信息
- 这是 world model 和 Sora 式生成模型的根本区别——不是"生成下一帧画面"，而是"预测状态发生了什么变化"

**World Model 与战略传播的对应关系：**

| World Model 概念 | 战略传播对应 |
|------------------|-------------|
| 当前状态 s | 目标受众当前的认知/态度/信念 |
| 候选动作 a | 可选传播策略/叙事框架/信息设计 |
| 结果预测 s' | "如果我用叙事框架 X，受众的认知会如何变化" |
| 最优选择 | 选择能最有效达到传播目标的策略 |

这个对应关系是整个第四章框架的理论基础。

### 2.3 Nye 公共外交的三层框架

**来源：** Nye, J. S. (2008). "Public Diplomacy and Soft Power." *The Annals of the American Academy of Political and Social Science*, 616(1), 94–109.

**三层框架回顾：**

| 层次 | 定义 | 决策逻辑 | 需要的 Agent 能力 |
|------|------|----------|-------------------|
| 日常沟通 | 对事件实时反应 | 反应式：发生了 X → 我应该回应 Y | 检索相关记忆 → 生成合理回应（生成式 agent 够用） |
| 战略传播 | 围绕简单、一致的信息持续传播数月到数年 | 预测式：受众现在的认知是 A → 我希望一年后变成 B → 我应该用什么叙事框架来推进这个转变 | 模拟不同叙事的长期传播效果 → 选择最优策略（**需要 world model**） |
| 关系建设 | 通过交流、奖学金等建立长期人际纽带 | 预测式：谁能成为未来十年对本国友好的关键人物 → 如何投资这些关系 | 模拟社会网络的长期演化 → 识别关键节点和杠杆点（**需要 world model**） |

**关键观察：** Nye 的框架不是三种不同的活动，而是三种不同的**时间尺度和决策逻辑**。日常沟通和战略传播的区别不是"发不发言"，而是"有没有一个关于受众认知变化的预测模型"。

**Nye 没说但隐含的命题：**
第二层和第三层的有效性取决于传播者能否准确预测"我的行动如何改变受众的认知"。这不是技术偏好，是逻辑必然——如果我不知道我的策略会产生什么效果，我就无法说我在"做策略"。

## 第三章：架构差距——为什么反应式 Agent 做不了战略传播

### 3.1 战略传播的任务结构

战略传播的核心不是一个"说话"任务，而是一个"决策"任务：

1. **目标设定：** "我希望目标受众一年后的认知/态度/行为是什么"
2. **策略空间枚举：** "我有哪几种可能的传播方案"
3. **效果预测：** "每种方案可能导致什么结果"
4. **策略选择：** "选效果最好的那个方案去执行"
5. **反馈修正：** "实际效果和预测有偏差，更新模型后重新预测"

其中步骤 3（效果预测）是战略传播区别于日常沟通的本质——你在脑中跑"如果我说 X，会发生 Y"的模拟。

### 3.2 生成式 Agent 能做什么、不能做什么

**能做：**
- 在合适的时机对合适的人说合适的话（反应式社交能力）
- 通过反思从大量经验中提炼洞察（"最近大家对 gentrification 话题很关注"）
- 通过计划分解实现短期目标（"这周要完成论文 → 今天写引言 → 现在开始写第一段"）
- 传播信息（传 party 消息、交流选举信息）

**不能做：**
- **不能模拟行动后果。** Agent 决定"邀请 Klaus 来 party"不是因为预测了这个邀请会改变 Klaus 的行为，而是因为"在类似情况下，人们会发邀请"——这是检索的结果，不是预测的结果。
- **不能比较策略。** Agent 不会想"我用三种不同的方式邀请 Klaus，哪种 Klaus 最可能来"——它一次性生成一个"合理的"邀请语。
- **不能追踪因果。** "Klaus 来了是因为我的邀请语打动了他"还是"是因为他没有其他安排"——agent 不区分这两种情况，也不积累关于"什么传播策略有效"的结构化知识。
- **不能做长期决策。** 因为每一步都是反应性的，没有全局目标函数来优化，agent 的行为是 emergent 的而不是 strategic 的。

### 3.3 这不是"做的不够好"，这是"架构假设不对"

关键论点：

> 生成式 agent 在战略传播任务上的局限，不是性能问题，不是"数据不够多""模型不够大""提示词不够好"，而是**它的架构假设排除了策略性决策所必需的前向预测组件**。

生成式 agent 的 architecture 隐含假设是：合理的行为来自对相关过去经验的充分检索和整合。但战略传播的决策逻辑是：合理的行为来自对不同可能未来的比较和选择。

这是两种不同的**决策本体论**。

| | 反应式决策 | 预测式决策 |
|---|---|---|
| 时间方向 | 过去→现在 | 现在→未来→现在 |
| 信息源 | 记忆流 | 世界模型 |
| 优化目标 | 无（做合理的事） | 有（最大化预期效果） |
| 策略表达 | 涌现 | 规划 |
| 理论对应 | 日常沟通（Nye 第一层） | 战略传播（Nye 第二、三层） |

### 3.4 一个示例来说明架构差距

**场景：** 国家 X 希望对国家 Y 的精英阶层传递一个信息："我们的经济改革是认真的，值得你们来投资。"

**生成式 agent 怎么做：** 检索和 Y 国精英过去的互动记忆 → 检索当前国际环境的相关信息 → 生成一个"在类似情况下看起来合理"的传播方案。比如"开一场新闻发布会，宣布经济改革措施"。合理吗？合理。但有策略性吗？不一定。

**World model agent 怎么做：** 
1. 建模 Y 国精英当前对 X 国的信念状态（"他们认为我们的改革不靠谱"）
2. 枚举可选传播策略：A = 新闻发布会，B = 邀请对方记者实地考察，C = 由第三方经济学家发布分析报告，D = A+B+C 组合
3. 预测每种策略对 Y 国精英信念状态的影响（"如果做 B，他们信任度可能从 20% 升到 45%""如果只做 A，可能几乎不变"）
4. 选择预期效果最好的策略
5. 执行后观测实际效果，更新世界模型

**区别不只是在效果上——区别在计算逻辑上。** 生成式 agent 是"做合理的事"，world model agent 是"做最有效的事"。日常沟通需要前者，战略传播需要后者。

## 第四章：预测性战略传播 Agent——一个融合框架

本章提出一个概念框架——不要求可运行代码，但必须有足够的清晰度让读者看到"这个东西怎么造"。

### 4.1 框架概览

我们提议的战略传播 agent 架构包含四个核心组件：

1. **受众世界模型（Audience World Model）：** 不是对所有事物建模——聚焦于对"受众的认知和信念如何响应信息输入"建模
2. **策略生成器（Strategy Generator）：** 给定目标和受众状态，生成候选传播策略（叙事框架、渠道选择、时机等）
3. **效果预测器（Outcome Predictor）：** 给定候选策略和受众当前状态，预测受众状态的变化。这是世界模型的核心——在潜在空间中做预测
4. **策略选择器（Strategy Selector）：** 比较不同策略的预测效果，选择最优策略

### 4.2 受众世界模型的形式化（简化版）

这是框架的核心。我们不是要训练一个实际的世界模型，而是要给出形式化定义，让读者知道这为什么和生成式 agent 不同。

- **受众状态 S(t)：** 在时间 t，目标受众对传播者（及传播者试图塑造的话题）的信念、态度和认知框架的向量表示
- **传播动作 A(t)：** 在时间 t，传播者选择的信息行动——叙事内容、渠道、受众细分、时机等
- **世界模型 W(S, A) → ΔS：** 给定当前受众状态 S 和传播动作 A，预测受众状态的变化 ΔS
- **策略 π(S, G) → A：** 给定当前状态 S 和传播目标 G，选择动作 A 使得目标函数最大化

**与生成式 agent 的根本区别形式化：**

- 生成式 agent 的决策：action = LLM(retrieve(memory_stream, observation), plan)
- World model agent 的决策：action = argmax_A [ expected_utility(W(S, A), goal) ]

前者是从过去中检索最相关的经验来指导一个合理的行动。后者是从可能未来中选择预期效果最好的行动。

### 4.3 为什么是潜在空间预测而非文本预测

这里必须说清楚我们说的 world model 是什么（以及不是什么）：

- **不是什么：** 我们不是在说"训练一个模型来逐字生成受众未来的反应文本"。那和 Sora 的像素生成属于同类错误——在原始数据空间做预测，计算上不可行也理论上不必要。
- **是什么：** 在潜在表征空间中学习"传播动作→受众信念变化"的映射。不需要知道受众"会说什么具体的话"，只需要知道"受众对我们某个话题的信任度会上升还是下降"以及"量的估计是多少"。

这对应 LeCun JEPA 的核心主张：预测应该在表征空间中进行，不是在原始信号空间中进行。

### 4.4 框架为什么适合战略传播

- **可追踪性：** 世界模型的状态更新是可审计的——你可以看"这条信息传入后，agent 认为受众信念发生了什么变化"，而不是面对 LLM 的黑箱输出
- **可干预性：** 如果受众世界模型给出的预测和实际不符，可以诊断是"状态表征学得不好"还是"策略生成有问题"——模块化架构使错误可定位
- **可积累性：** 每次传播行动后更新世界模型（"我以为这个叙事框架有效，实际效果偏差很大，更新模型"）——这是生成式 agent 完全不做的事情

### 4.5 与 Nye 框架的映射

| Nye 层次 | 框架中的组件 | 说明 |
|----------|-------------|------|
| 日常沟通 | 策略生成器（快速模式） | 世界模型提供当前受众状态，文本 LLM 生成合理回应——这是"增强型反应式"，不是纯预测式 |
| 战略传播 | 世界模型 + 效果预测器 + 策略选择器 | 全栈运行：模拟多种策略→比较预期效果→选择最优→执行后更新模型 |
| 关系建设 | 世界模型（长期演化模式） | 需要学习"关系如何随时间和互动演化"的长期动力学，而不是单步效果预测 |

## 第五章：国家安全含义

本章回答"这对国家安全有什么用"。分三个层面。

### 5.1 认知域作战的建模工具

认知域作战是现代混合战争的核心——不仅争夺领土，争夺人心。中国、俄罗斯和美国都在大力投资基于信息的影响力行动。

现有认知域作战研究的主要方法是案例分析和事后复盘。问题在于：你只能在事情已经发生之后去分析，无法在策略实施之前进行可控的"what if"推演。

我们的框架提供**基于模型的战略传播推演**——这不是说 AI 替代人来决策，而是说：
- 可以在实验环境中测试"如果采用叙事框架 X 而非 Y，效果差异会多大"
- 可以进行反事实分析："如果我们当时没有选择叙事 A，对方会怎么反应"
- 可以让分析者和决策者在部署实际传播策略之前，先看到模型预测的效果差异

### 5.2 大国战略竞争的模拟工具

两个大国在第三方国家争夺影响力的场景——这是国际新秩序的典型现象。当前的模拟工具（包括 agent-based modeling 和系统动力学模型）依赖研究者手工预设 agent 的行为规则——"如果 agent 收到消息 A，则以概率 p 执行行为 B"。这忽略了战略传播的动态性和情境依赖性。

World model agent 的优势：
- 不预设固定规则，而是学习"什么样的信息在什么条件下改变受众信念"
- 能模拟大国博弈中的**递归推理**——"我知道你知道我知道，所以我的传播策略要考虑你的反制"
- 能模拟**时间不对称**——"我今天投入的传播资源可能在三个月后产生效果"

### 5.3 从"AI 威胁论"到"AI 方法论"

学界和媒体关于 AI 与国家安全的主流叙事是威胁性的：deepfakes 会摧毁真相、算法会操纵选举、AI 武器化会失控。

我们提出一个不同但互补的叙事：**AI 仿真可以成为研究国家安全的科学工具，但前提是你选对了架构。** 反应式生成 agent 可以模拟信息在社交网络中的传播，但无法模拟传播策略的因果效果。World model 补齐这个缺口。

这是"AI 赋能国家安全研究"而不是"AI 威胁国家安全"——对国家安全的学术建设有正面意义。

## 第六章：未来工作——实验验证蓝图

本章预埋完整实验设计。论点能否成立，最终取决于世界模型 agent 是否在实验中优于生成式 agent。

### 6.1 实验目标

验证核心假设：**在战略传播任务中，具有世界模型的 agent 比反应式生成 agent 更能选择和执行有效的传播策略。**

形式化假设：
- H1（策略选择能力）：给定多个传播策略选项，world model agent 选择的策略优于生成式 agent 选择的策略（更高的受众态度改变量）
- H2（长期效果）：在多轮传播互动中，world model agent 的累计传播效果好于生成式 agent（累积奖励曲线更高）
- H3（自适应）：当受众偏好发生变化时，world model agent 调整策略的速度快于生成式 agent

### 6.2 实验场景

**基本设定：** 选择一个可操作化的地缘政治传播场景。建议：

> 国家 A（传播者）试图影响国家 C 精英阶层对国家 B（竞争对手）的投资政策态度。场景中 3 个"国家 agent"（A、B、C 官方）+ 3 个"媒体 agent"（亲 A、亲 B、中立）+ 2 个"精英 agent"（国家 C 的商业领袖和政策制定者）。共计 8-10 个 agent。

**任务：** 在 10 轮传播回合中，国家 A 的 agent 选择传播策略（直接声明、第三方背书、议程设置、叙事框架重构等），精英 agent 根据自己的世界模型更新信念并表达态度。

### 6.3 实验条件

| 条件 | 传播者 agent | 受众 agent | 说明 |
|------|------------|-----------|------|
| Baseline | 生成式 agent | 生成式 agent | 所有 agent 都是反应式架构 |
| Partial | World model agent | 生成式 agent | 只有传播者有世界模型，受众仍然是反应式的 |
| Full | World model agent | World model agent | 双方都有世界模型——最高保真度 |

### 6.4 关键指标

- **主要指标：** 受众信念变化（目标维度的态度评分变化量，由独立 LLM 评委评估）
- **次要指标：** 策略多样性（agent 是否探索了多种策略选项）、策略一致性（agent 的策略是否围绕一个核心叙事展开）、预测准确度（world model agent 的预测与实际结果之间的差距）
- **过程指标：** 每轮 world model agent 在策略选择时实际调用了多少次"模拟"（效果预测器的运作频率）

### 6.5 技术实现路径

略过无需在此详述。已知约束：一张 A800 GPU。方案：(1) 使用小参数量开源模型（Qwen2.5-7B 级别）驱动 agent 的自然语言能力；(2) 世界模型组件可以是一个轻量的独立预测模块（不需要大模型来跑预测）；(3) 或使用云端 API 服务绕开本地算力限制。

## 第七章：结论

### 7.1 核心论点回顾

- 战略传播需要效果预测和策略选择能力
- 当前生成式 agent 只有反应能力，没有预测能力——这不是 bug，是架构假设使然
- World model 的计算框架（前向预测 + 基于模型的规划）填补了这个 gap
- Nye 的公共外交三层框架为这个架构差距提供了理论锚点——反应式够用第一层，预测式需要支撑第二、三层

### 7.2 理论贡献

1. 首次识别并形式化生成式 agent 架构在战略传播中的结构性局限
2. 提出 World Model 增强的战略传播 agent 概念框架
3. 在计算架构和传播理论之间建立了可检验的对应关系

### 7.3 局限与边界

- 这是框架论文，没有实证验证（但提供了可操作的实验蓝图）
- 实际训练世界模型面临数据稀缺问题（"传播效果"的数据不易获取）
- 框架聚焦于"认知/态度"层面的传播效果，不考虑情绪、身份认同等更难建模的维度
