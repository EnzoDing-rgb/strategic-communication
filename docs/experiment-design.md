# 实验设计蓝图

## 研究问题

装了 World Model 的 Agent，在战略传播任务中是否优于普通生成式 Agent？

## 场景

3 个国家：

| 角色 | 说明 |
|------|------|
| A 国 | 我方，有两个 Agent 版本 |
| B 国 | 对手，行为固定（基准线） |
| C 国 | 目标受众，其精英态度可被影响 |

3 个媒体渠道：官媒、社交媒体、学术渠道。

## Agent 配置

| Agent | 架构 | 能力 |
|-------|------|------|
| 普通 Agent | 生成式（Stanford 小镇架构） | 观察→检索记忆→反思→生成反应 |
| World Model Agent | 生成式 + 预测模块 | 同上 + 推演候选策略效果→选最优 |
| B 国 Agent | 固定策略 | 维持基准对手行为 |

## 流程

1. 初始化 C 国精英对 A 国的态度为「警惕」
2. 进行 10 轮传播，每轮：
   - A 国 Agent 选择叙事策略
   - 选择投放渠道
   - B 国 Agent 同时投放对手叙事
   - C 国精英更新态度
3. 两种 Agent 各跑一次，其他条件完全相同

## 衡量指标

- **主指标**：C 国精英对 A 国的态度分（负面→中立→正面），10 轮后的绝对偏移量
- **对比**：普通 Agent 偏移量 vs World Model Agent 偏移量

## 预期结果

World Model Agent 的受众态度偏移应该显著大于普通 Agent。因为它在每轮都做了效果预测和策略选择，普通 Agent 只是在做「合理」的反应。

## 计算成本

一张 A800，预计几小时。不需要大规模集群。

## 备注

这是一个概念验证实验——不追求生态效度，只验证「预测式架构在原理上优于反应式架构」。生态效度留到后续工作。
