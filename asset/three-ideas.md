# 三个研究方向（课程论文 brainstorm 产出）

日期：2026-06-11

---

## Idea 1 · 传统多智能体仿真

用 LLM-based agent（如 Stanford 小镇架构）在虚拟环境中模拟战略传播。

**做法：**
- 多个 agent 各自维护 memory stream
- 事件发生 → agent 观察 → 检索记忆 → 反思 → 规划 → 生成行动
- 观察 agent 之间的信息传播和态度变化

**问题：**
- Agent 是反应式的：遇到什么回应什么
- 不预测行为的后果
- 不比较「方案 A vs 方案 B 哪个效果更好」
- Stanford 论文自己报告的局限：记忆不完整、添油加醋、从不拒绝、知识泄露

**结论：** 反应式架构够处理日常沟通（小时/天），够不着战略传播（月/年）。

## Idea 2 · 世界模型增强（选中，论文核心）

给 multi-agent 系统加一个预测组件（World Model），让 agent 在行动前先模拟多种策略的效果。

**做法：**
- Agent 维护世界模型 f(state, action) → predicted_next_state
- 面对传播任务时，生成多个候选策略
- 分别预测每个策略的传播效果
- 选效果最优的策略执行

**核心贡献：**
- 指出生成式 agent 的反应式局限
- 提出世界模型增强框架
- 用 Nye 三层框架论证理论必要性

**与 LLM 的本质区别：**
- LLM：预测下一个 token → 生成「听起来像分析」的文本
- World Model：预测下一个状态 → 学习状态转移函数，做因果推演
