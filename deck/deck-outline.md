## Slide 2：问题——什么是战略传播？为什么 AI 可以研究它？

**先说什么是战略传播（人话版）：**

- 一个国家对另一个国家的民众说话，不是为了聊天，是为了改变对方的看法
- 不是发一条推文、开一场记者会——那是"日常回应"
- 真正的战略传播是：*我有六个月时间，我要让某个国家的精英阶层相信"和我们合作是安全的"。我应该说什么、通过什么渠道说、按什么节奏说？*

**这和 AI 有什么关系？**

- 如果 AI agent 能在模拟环境中扮演国家 A（传播方）、国家 B（受众方）、国际媒体（中介方），我们就能**在部署真实传播策略之前，看到不同策略的效果**
- 等于给战略传播装了风洞实验室

**但有个问题：** 当前的 AI agent 架构缺少做这件事的关键能力。这个能力叫**预测**。

---

## Slide 3：当前 AI Agent 怎么做决策？

**以斯坦福小镇（Park et al., 2023）为例：**

agent 面对一个情况时的决策流程——

```
1. 观察：发生了什么？
2. 检索记忆：我过去在类似情况下做过什么？
3. 反思：从大量记忆中提炼一些洞察
4. 生成反应：用大语言模型生成一个"合理的"回应
5. 执行
```

**翻译成人话：**

agent 做的事 = 遇到情况 → 翻翻过去的经验 → 做出一个看起来合理的反应

**这个架构能做什么：**
- 日常社交：聊天、传消息、约人吃饭
- 短期任务：今天要做什么，拆成几个步骤

**这个架构不能做什么：**
- 不能想"如果我这样说，对方会怎么反应"
- 不能比较"三种不同的说法，哪一种效果最好"
- 不能从"我的策略失败了"中学到"什么策略更有效"

**为什么不能？** 因为它的决策只依赖**过去的记忆**，不依赖**对未来的预测**。

## Slide 4：少的那块拼图——世界模型（World Model）

**什么是世界模型？**

用你熟悉的 CS 语言：

> 世界模型就是一个状态转移函数 f(state, action) → next_state。Agent 在执行动作之前，先用这个函数在脑子里模拟一遍结果。

**和 LLM 的 next-token prediction 有什么区别？**

| | LLM | World Model |
|---|---|---|
| 预测什么 | 下一个 token | 下一个状态 |
| 预测几次 | 一次 | 多次（每个候选动作都预测一次） |
| 有没有选择 | 没有，直接输出 | 有，预测完选最好的 |

**具体的：**

没有世界模型的 agent：
```
"有人找我投资" → 检索记忆："以前有人找我投资时我说了什么" → 生成回复
```

有世界模型的 agent：
```
"有人找我投资"
  → 方案 A："强调高回报率" → 预测：对方可能觉得太冒险
  → 方案 B："强调安全性" → 预测：对方态度从"观望"变"有兴趣"
  → 方案 C："强调市场趋势" → 预测：效果一般
  → 选 B，执行
```

**就是这么简单。** 世界模型 = 在做决定之前，多跑几次模拟。

## Slide 5：为什么战略传播非要有预测不可？

**约瑟夫·奈（Joseph Nye）的公共外交理论——三层框架：**

奈是"软实力"概念的提出者。他把一个国家的对外传播分成三层：

| 层次 | 做什么 | 需要什么能力 |
|------|--------|-------------|
| 第一层：日常沟通 | 对突发事件做回应 | 检索过去经验，生成合理回应 |
| 第二层：战略传播 | 围绕一个核心信息持续影响目标受众 | **需要预测：哪种叙事框架效果最好** |
| 第三层：关系建设 | 几十年的长期人际关系投资 | **需要预测：谁会成为未来的关键人物** |

**奈没说但隐含的逻辑：**

你如果要做第二层和第三层，你不能只是"做合理的事"。你必须能回答一个问题："如果我做 X 而不是 Y，半年后受众的态度会有什么不同？" 这个问题只能用世界模型来回答。

**我们的论点：**

> 生成式 agent 的架构可以做第一层，但做不了第二层和第三层。不是因为它不够好——是因为它缺了一个组件。这个组件就是世界模型。

---

## Slide 6：我们提出的框架 + 国家安全含义

**框架很简单——给 agent 加一个组件：**

```
当前生成式 agent：
  观察 → 检索记忆 → 生成反应

我们提出的：
  观察 → 检索记忆 → [世界模型：预测不同动作的效果] → 选择最优动作 → 执行
```

不是替换，是增强。方括号里的是新加的。

**这对国家安全有什么用？**

1. **认知域作战可以有实验工具。** 不只靠案例分析，可以跑仿真看不同策略的效果差异。
2. **大国竞争可以模拟。** 两个大国在第三国争夺影响力，世界模型 agent 可以模拟多轮博弈。
3. **政策制定前可以先推演。** 不是"AI 替人决策"，是"AI 告诉人不同决策可能导致什么结果，人再做选择"。

**一个具体的未来实验（论文不跑，但设计好）：**

- 3 个国家 agent + 3 个媒体 agent + 2 个精英 agent
- 对比：普通 agent 做战略传播 vs 加了世界模型的 agent 做战略传播
- 看哪边能在 10 轮内更有效改变目标受众的态度

## Slide 7：总结

**一句话：**

> 战略传播的本质是"效果预测 + 策略选择"。当前 AI agent 能做前者，但缺后者。缺的那块叫世界模型。

**三个贡献：**
1. 指出了当前 AI agent 架构在战略传播上的结构性缺陷（不是性能问题，是架构问题）
2. 提出了加入世界模型的增强框架
3. 用 Nye 的三层公共外交理论论证了为什么这个增强是必要的

**一个开放问题：**
世界模型在文本/社会交互领域怎么训练？这是未来工作的核心。
