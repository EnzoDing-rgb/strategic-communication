理论锚点：约瑟夫·奈的公共外交框架

三层公共外交 = 三种预测深度

Nye 以「软实力」闻名。公共外交是软实力的操作层，三层框架是操作指南。我们盯第二、三层。

Layer 1

日常沟通

外交部回应记者提问，对事件实时表态
时间：小时 / 天
做法：检索已有立场，生成前后一致的回应
预测需求：几乎为零
→ LLM 够用
Layer 2

战略传播

围绕一个核心叙事，持续影响他国精英
时间：月 / 年
做法：设计叙事 → 选择渠道 → 持续投放 → 等半年民调才知道效果
预测需求：必须事前推演
→ 需要 World Model
Layer 3

关系建设

学术交流、文化项目，培养长期友好网络
时间：年 / 十年
做法：A 影响 B，B 影响 C，反馈循环交织
预测需求：效果几十年才显现，中间无数人互相影响
→ 需要 World Model

Nye 的每一层，天然定义了一个预测问题——只是他没这么说

本文贡献：把 Nye 的经验框架翻译成计算语言 → 越往上层，越不可能靠「反应」，越需要「预测」
